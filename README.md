# Jahreszeiten-Simulator (Season Simulator)

A simple interactive tool to visualize the Earth's orbit around the Sun and understand how seasons change across both hemispheres.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.x-green.svg)

## Description
This tool provides a simplified 3D/2D visualization of the Earth's movement around the Sun every year. It helps users to visualize the tilt of the Earth's axis and how it affects sunlight distribution throughout the year - which causes the seasons.

## Key Features
- **Dynamic Orbit:** Watch the Earth move around the Sun in real-time. The corresponding date is shown in the viewport.
- **Show Today:** Move the simulation to the actual current position of the Earth.
- **Velocity Control:** Adjust the speed of both Earth's rotation and its orbital movement.
- **Reference Lines:** Toggle visibility for the Equator, Tropics, and Arctic Circles.
- **Hemisphere Toggle:** Switch descriptions to focus on either the Northern or Southern Hemisphere perspective.
- **Language Support:** Toggle between German and English interface.
